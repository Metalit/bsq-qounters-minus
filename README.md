# Qounters-

Cheap knockoff of Counters+ for Quest

## Implemented Qounters
- Cut
- Miss
- Notes
- NotesLeft
- Spinometer
- Speed
- Score
- PB

## Unimplemented Qounters
- Fail
- Progress
